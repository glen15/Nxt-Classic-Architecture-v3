# 2. RandomTextApp - 업데이트 미션

## 클라이언트 업데이트

### 미션: 앱 타이틀 변경

- 현재: `확신없는 랜덤 명언`
- 변경: `확신 넘치는 랜덤명언 by 홍길동`

> `홍길동` 부분에 자신의 이름을 넣으세요.

**힌트:** `client/src/App.tsx`

---

## 서버 업데이트

### 미션 1: 포트를 80으로 변경

- 서버가 80번 포트에서 실행되도록 변경하세요.

**힌트:** `server/.env`

### 미션 2: 명언 접미사 변경

- 현재: 명언 저장 시 `...아마도...` 가 자동으로 붙음
- 변경: `...!!!...` 가 붙도록 수정

**힌트:** `server/src/server.ts`

### 미션 3: 확신 있게 / 확신 없게 저장 버튼 분리

- 현재: 명언 저장 버튼이 1개
- 변경: `확신 있게 저장` 버튼과 `확신 없게 저장` 버튼 2개로 분리
- 확신 있게 저장하면 `...!!!...` 가 붙고, 확신 없게 저장하면 `...아마도...` 가 붙도록 구현하세요.

**힌트:** 클라이언트와 서버가 이 차이를 어떻게 구분할지 직접 설계해보세요.

---

## 주의

> 명시적인 요구사항에 따라 발생하는 추가 변경사항들을 잘 확인하세요.

> 접미사 변경은 **새로 저장되는 명언**에만 적용됩니다. 기존 데이터의 `...아마도...`는 변경되지 않습니다. 기존 데이터를 수정하는 것은 완전히 다른 작업입니다.
