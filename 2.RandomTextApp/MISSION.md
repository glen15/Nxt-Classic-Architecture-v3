# 2. RandomTextApp - 업데이트 미션

## 클라이언트 업데이트

### 미션: 앱 타이틀 변경

- 현재: `확신없는 랜덤 명언`
- 변경: `확신 넘치는 랜덤명언 by 000`

> `000` 부분에 자신의 이름을 넣으세요.

**수정 파일:** `client/src/App.tsx`

---

## 서버 업데이트

### 미션 1: 포트 변경

- 현재: `8000`
- 변경: `80`

**힌트:** `server/.env` 파일에서 `SERVER_PORT` 값을 수정하세요.

> 80번 포트는 HTTP 기본 포트이므로 주소에서 포트 번호를 생략할 수 있습니다.
> 변경 후 클라이언트의 `VITE_SERVER_URL`도 함께 수정해야 합니다.

### 미션 2: 명언 접미사 변경

- 현재: 명언 저장 시 `...아마도...` 가 자동으로 붙음
- 변경: `...!!!...` 가 붙도록 수정

**수정 파일:** `server/src/server.ts`

---

## 확인 방법

1. 브라우저에서 앱 접속 → 타이틀에 자기 이름이 보이는지 확인
2. 새로운 명언을 저장 → `...!!!...` 가 붙는지 확인
3. 서버가 80번 포트에서 실행되는지 확인
